# 📊 Meta Ads Performance Dashboard

![Power BI](https://img.shields.io/badge/Power%20BI-Dashboard-F2C811)
![Status](https://img.shields.io/badge/Status-Completed-brightgreen)
![Business Intelligence](https://img.shields.io/badge/Project-Business%20Intelligence-blue)

An interactive **Power BI dashboard** designed to analyze and monitor advertising performance across **Meta platforms (Facebook & Instagram)**. The dashboard provides marketers with real-time insights into campaign effectiveness through KPI tracking, demographic analysis, geographic visualization, and advertising performance metrics.

---

# 📌 Project Objective

The primary objective of this dashboard is to help digital marketers evaluate advertising campaign performance and optimize marketing strategies using data-driven insights.

The dashboard enables users to:

- Monitor campaign performance
- Track customer engagement
- Analyze advertising ROI
- Compare ad formats
- Understand audience demographics
- Improve marketing decisions

---

# 📊 Dashboard Overview

The dashboard summarizes the overall performance of Meta advertising campaigns using multiple Key Performance Indicators (KPIs).

### Key Metrics

- 📢 Impressions
- 🖱 Clicks
- 👍 Shares
- 💬 Comments
- 🛒 Purchases
- ❤️ Engagements
- 📈 Engagement Rate
- 🎯 Click Through Rate (CTR)
- 🔄 Conversion Rate
- 💰 Purchase Rate
- 💵 Total Budget
- 📊 Average Campaign Budget

---

# 📈 Dashboard Features

## 📌 KPI Cards

Quick overview of campaign performance through dynamic KPI cards displaying:

- Total Impressions
- Total Clicks
- Engagements
- Purchases
- Campaign Budget
- Conversion Metrics

---

## 👥 Audience Analysis

Analyze user engagement based on:

- Gender Distribution
- Age Groups

This helps identify which audience segments generate the highest engagement and clicks.

---

## 🌍 Geographic Analysis

Interactive world map displaying campaign performance by country.

Provides geographic insights for identifying high-performing regions and audience locations.

---

## 📅 Time-Based Analysis

Monitor campaign performance over time using:

- Monthly Analysis
- Weekly Trends
- Hourly Trends

These visualizations help identify peak engagement periods and optimize campaign scheduling.

---

## 📢 Ad Type Performance

Compare advertising formats including:

- Carousel Ads
- Image Ads
- Stories
- Video Ads

Performance metrics include:

- Impressions
- Clicks
- CTR
- Engagement Rate
- Purchase Rate
- Conversion Rate

---

## 🎛 Interactive Filters

The dashboard includes slicers for dynamic analysis:

- Dynamic KPI Selection
- Campaign Name
- Month
- Target Interests
- Platform Selection (Facebook / Instagram)

These filters allow users to customize the dashboard for detailed campaign analysis.

---

# 📊 Business Insights

The dashboard helps answer business questions such as:

- Which advertising campaign performs the best?
- Which age group generates the highest engagement?
- Which gender has the highest click rate?
- Which country contributes the most traffic?
- Which ad format delivers the highest conversion rate?
- What are the peak engagement hours?
- How efficiently is the advertising budget utilized?
